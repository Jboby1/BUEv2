General

All in-game Attack/Defense Capture Point maps now use the prefix "AD_"

Any A/D CP maps added in the future will have their names changed to match this

Valve now hosts a small number of Arena-exclusive servers to gauge popularity. If popular enough, more will be added

Added the Bolt-Action Blitzer, Black Rose, Flamingo Kid, and Triclops to the drop pool, as the games they could be achieved from have shut down.

Copies of these items obtained before this update are now Genuine quality, and tradable

Revamped inventory system with new options including multi-select, mass deletion, and additional/smarter auto-sort options

Added "New Zealium" weapons, which can be obtained from a brand-new Chemistry Set by inputting 100 Refined Metal and a class token
